## Structured Data Priorities

### Recipe Pages (HIGH PRIORITY)
Each recipe page should have:
```json
{
  "@context": "https://schema.org",
  "@type": "Recipe",
  "name": "{recipe name in page language}",
  "description": "{howIsMade excerpt}",
  "recipeIngredient": ["{ingredient1}", "{ingredient2}"],
  "recipeInstructions": "{howIsMade full text}",
  "recipeYield": "{servings} servings",
  "recipeCategory": "{category}",
  "recipeCuisine": "{origin country}",
  "image": "https://weeklymeaplanner.com/images/{slug}.jpg"
}
```

### BreadcrumbList
```json
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {"@type": "ListItem", "position": 1, "name": "Home", "item": "/en/"},
    {"@type": "ListItem", "position": 2, "name": "Recipes", "item": "/en/recipes/"},
    {"@type": "ListItem", "position": 3, "name": "{Recipe Name}"}
  ]
}
```

## hreflang Implementation

Every page must have `<link rel="alternate" hreflang="{lang}" href="{url}">` for all 14 languages + `x-default` pointing to `/en/`.

Verify in `scripts/generate-content.mjs` that this is generated correctly.

## Internal Linking Strategy

### Current weakness
Recipe pages are islands — no links to related recipes.

### Fix
In `scripts/generate-content.mjs`, for each recipe page:
1. Pick 3-5 recipes from same `category` or same `origin`
2. Add "You might also like" section at bottom of page
3. Link text = recipe name in page language

### Homepage links
- Featured recipe of the week (rotates automatically by week number)
- "Popular recipes" section linking to top 6 recipes by category

## Title & Description Templates

### Recipe page
- Title: `{Recipe Name} — {Category} Recipe from {Origin} | Meal Planner`
- Description: `Learn how to make authentic {Recipe Name}. {First sentence of howIsMade}. Free weekly meal planning.`

### Recipes index
- Title: `{N} Recipes from Around the World | Weekly Meal Planner`
- Description: `Browse {N} authentic recipes in {language}. Plan your week, generate a shopping list, download a PDF.`

### Homepage
- Title: `Weekly Meal Planner — Plan Your Meals in Minutes | Free`
- Description: `Plan a full week of meals from 175 global recipes. Generate your shopping list automatically. Free PDF download. Available in 14 languages.`
