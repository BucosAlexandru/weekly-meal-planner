バンデハ・パイサ

バンデハ・パイサはコロンビアの国民食であり、ラテンアメリカで最も豪快な一皿です——フリホーレス（赤いんげん豆）、白ご飯、カルネ・モリダ（挽き肉）、チチャロン（揚げた豚の皮）、チョリソ、モルシーラ（ブラッドソーセージ）、目玉焼き、アレパ、揚げたプランテン、アボカド、パンの9-11種類が一枚の皿に盛られます。 アンデス山脈のアンティオキア地方が起源で、農場労働者が一日中の肉体労働に必要なエネルギーを得るために食べていました。

本格的な味

材料 4人前

  • フリホレス・アンティオケニョス用：乾燥赤キドニーまたはカーガマント豆400g、一晩浸す
  • 中サイズ人参1本（~100g）すりおろし + 中サイズトマト2個（~250g）すりおろし + 玉ねぎ1個（~150g）みじん切り + にんにく4片、ホガオ用
  • 豆用：油大さじ2 + トマトペースト大さじ1 + クミン小さじ1 + 塩小さじ1
  • 熟したプランテン1本（~150g）角切り、豆用
  • アロス・ブランコ用：中粒米360g（2カップ）+ 水540ml + 塩小さじ0.5
  • カルネ・モリダ用：牛ひき肉500g + みじん切り玉ねぎ1個 + にんにく4片 + すりおろしトマト2個 + 油大さじ2 + クミン小さじ1 + 塩小さじ1 + コショウ小さじ0.5 + 牛肉ストック50ml
  • チチャロン用：皮付き豚バラ肉800g、~10cmの8本に切る + 塩小さじ1
  • 生コロンビアチョリソ4本（各~80g）
  • モルシージャ4本（血ソーセージ、各~80g）
  • 小さなアレパ4枚用：プリクック白コーンミール（Harina P.A.N.）200g + 温水240ml + 塩小さじ0.5 + 鍋用の油大さじ1
  • マドゥロ用：とても熟したプランテン2本（皮が真っ黒）+ 油大さじ2
  • 目玉焼き用の大きな卵4個 + 油大さじ1 + 塩とコショウ
  • 完熟アボカド2個、半分に切って種を取る
  • カリカリの白パン4切れ + ライムくし切り4個 + 提供用の塩

作り方

  1. 1浸す＆計画：赤豆400gを一晩浸す、バンデハ・パイサは6つの並行コンポーネントを必要とする——開始前に全ての材料と道具を集め、3つの鍋と2つのフライパンを用意する.
  2. 2豆：豆を水切りし、1.5Lの水でおろし人参+おろしトマト1個+玉ねぎ+にんにく2片+クミン小さじ1+塩小さじ1と60–90分柔らかくなるまで煮込む、最後の15分でホガオを作る：油大さじ2で残りの玉ねぎ+にんにく2片+おろしトマト1個+トマトペースト大さじ1を5分炒める、豆に角切りプランテンと一緒に加え、さらに10分煮込む.
  3. 3ご飯：360gの米を冷水で3回洗う、540mlの水と塩小さじ0.5と一緒に鍋に入れる、沸騰させ、蓋をして最弱火に12分、火を止めて10分蒸らす.
  4. 4カルネ・モリダ：フライパンに油大さじ2を中強火で熱す、玉ねぎとにんにくを5分柔らかくなるまで炒める、牛ひき肉500gを加え、スプーンで崩しながら8分焼く、おろしトマト2個、クミン小さじ1、塩小さじ1、コショウ小さじ0.5を加え5分煮る、ストック50mlでデグラーズする.
  5. 5チチャロン：皮付き豚バラ8本を200ml冷水と塩小さじ1の鍋に入れる、沸騰させ、蓋をして30分煮込む、完全に水切り、同じ鍋で豚肉を自身の脂で中強火8–10分、頻繁に返しながら、皮が金茶色でカリカリになるまで焼く、（代替：200 °Cオーブンで40分、ラックの上で、）.
  6. 6アレパ：ボウルで250gコーンミール、240mlの温水、塩小さじ0.5を湿った生地になるまで混ぜる、5分休ませる、直径10cm、厚さ1cmの4枚のパティに成形する、熱いフライパンに油大さじ1を加え中火で片面4–5分、金茶色で暗い斑点ができるまで焼く.
  7. 7マドゥロ+卵+ソーセージ：黒プランテン2本を1cm厚の輪切りに、油大さじ2で中火2分ずつ片面、キャラメル化するまで揚げる、別のフライパンで生チョリソ4本を片面5分、モルシージャ4本を片面3分焼く、それから油小さじ1で卵4個を2分、白身が固まり黄身が流れる状態に焼く.
  8. 8盛り付け：4枚の大きな皿にこの順序で並べる：ご飯の山、豆の盛り、カルネ・モリダ一人前、チチャロン2片、チョリソ1本、モルシージャ1本、肉の上に目玉焼き1個、マドゥロ2スライス、アボカド1/2、アレパ1枚、パン1切れ、ライム1切れ、すぐ提供——バンデハ・パイサはナイフとフォークで食べ、一口ごとに複数のコンポーネントを組み合わせる.
