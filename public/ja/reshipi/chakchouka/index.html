シャクシュカ

シャクシュカは北アフリカ料理の象徴的な料理であり、チュニジアが発祥地です。スパイスを効かせたトマトとパプリカのソースでポーチされた卵は、共同の朝食の象徴となりました。食材のシンプルさが、驚くほどの感覚的な複雑さを隠しています。 チュニジアからレシピは北アフリカ全土、イスラエル、中東へと広まり、地元の風味と辛さに適応していきました。各家庭はそれを自分たちのものと考え、少し異なるアレンジで作ります。

材料 2人前

  • 室温の大きな卵4個
  • エクストラバージンオリーブオイル大さじ4（チュニジア産または地中海産）
  • 中ぐらいの黄玉ねぎ1個（約150g）、細かく刻む
  • マイルドな赤パプリカ2個（約300g）、種を取って1cm角に切る
  • 小さな赤唐辛子1本を細かく刻むかチュニジアのハリッサペースト小さじ1（より伝統的な選択）
  • ニンニク4片、みじん切り
  • クミンパウダー小さじ1（kamoun）— 温かい香辛料のベース
  • キャラウェイパウダー小さじ1（karwiya）— チュニジアを定義する温かい香辛料、見た目が似ているがクミンとは別物；入手できない場合はフェンネルシードで代用
  • 甘いパプリカ小さじ1
  • コリアンダーパウダー小さじ0.5
  • 缶詰の砕いたトマト400g（または皮をむいてすりおろした熟したトマト4個）
  • コクのためのトマトペースト大さじ1
  • 塩小さじ1+挽きたて黒胡椒小さじ0.5+酸味のバランスのための砂糖ひとつまみ
  • フェタタイプのチーズ60g（チュニジア風またはブルガリア風、崩したもの、任意）+ 新鮮なイタリアンパセリ + コリアンダー + レモンのくし切り + 新鮮なバゲットまたはチュニジアのkhobz taboumaを添えて

作り方

  1. 1芳香材料の準備：チャクチューカは早く調理される — mise-en-placeは交渉の余地がない。玉ねぎ150g + 赤パプリカ300g + 赤唐辛子1本を角切りにする。ニンニク4片をみじん切りにする。スパイスのカルテット — クミン + キャラウェイ + パプリカ + コリアンダー — を小さなボウルに計量する。キャラウェイがこれをチュニジア風にし、イスラエルまたはレバント風のシャクシューカではなくする。 2.
  2. 2油を温める：重い24cmスキレット（鋳鉄が理想 — 陶器のタジン鍋が最も伝統的なチュニジア風）でオリーブオイル大さじ4を中火で揺らめくまで熱する。 3.
  3. 3ソフリット：刻んだ玉ねぎ150gとひとつまみの塩を加え、非常に柔らかく半透明になるまで6–8分炒める — 茶色にしない。チュニジア風チャクチューカは、ゆっくり汗をかいた玉ねぎの甘さを求め、キャラメル化ではない。 4.
  4. 4パプリカ + 唐辛子 + ニンニク：角切りの赤パプリカ300g + 刻んだ赤唐辛子1本（またはハリッサ小さじ1）を加える。パプリカが柔らかくなるが鮮やかな赤を保つまで6–8分炒める。みじん切りニンニク4片を加え、香りが立つまで1分炒める。 5.
  5. 5スパイスを開花させる：スパイスミックス（クミン小さじ1 + キャラウェイ小さじ1 + パプリカ小さじ1 + コリアンダー小さじ0.5）を熱い油と野菜のベースに直接加える。強く香り立つまで30秒かき混ぜる — 焦がさない、さもないと料理が苦くなる。 6.
  6. 6ソースを作る：トマトペースト大さじ1を混ぜ入れ、1分炒める。砕いたトマト400g + 塩小さじ1 + 胡椒小さじ0.5 + 砂糖ひとつまみを注ぐ。時々かき混ぜながら8–10分煮込み、ソースが濃くなるまで — 表面をスプーンでなぞった跡がすぐに埋まらない明確な道を残すべき。 7.
  7. 7卵を加える：スプーンの背で、煮立っているソースに4つのくぼみを作る。各くぼみに卵1個を割り入れる — 黄身を無傷に保つ。白身にひとつまみの塩、黄身に胡椒のひねりをかける。スキレットに蓋をして、中弱火で4–6分加熱する。白身が固まりかけているが黄身は流動的なまま — 蓋越しに覗く；白身は不透明で、黄身はまだ鮮やかなオレンジ色で震えているべき。 8.
  8. 8仕上げて提供する：上に崩したフェタ60g + 刻んだパセリとコリアンダーをたっぷりひと握り散らす。フライパンを直接テーブルに持ってくる — チャクチューカは常に共同で、フライパンから食べる。各食卓の客は新鮮なバゲットまたはkhobz taboumaの一片をちぎり、それを使ってソースを拭き取り、黄身を割る。自分の取り分にレモンのくし切りを絞る。調理から10分以内が最高、黄身がまだ溶けている間。.
💡
ポイント: 提供前に味見して、塩と調味料を好みに合わせて調整する。
